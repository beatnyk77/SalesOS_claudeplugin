# 🛒 E-Commerce Seller Toolkit for Claude Code

**The complete AI-powered workflow toolkit for Amazon FBA & DTC e-commerce sellers.**

Stop spending hours writing listings, responding to reviews, and building ad campaigns from scratch. This plugin gives you 7 expert slash commands + 1 intelligent subagent — built on deep e-commerce expertise, not generic AI templates.

| Command | What It Does | Time Saved |
|---|---|---|
| `/product-listing` | Amazon + Shopify listings (title, bullets, description, backend keywords) | 2–3 hrs → 2 min |
| `/review-response` | Professional responses to 1–5 star reviews | 15 min → 30 sec |
| `/competitor-teardown` | Positioning gaps + copy angles vs any competitor | 3 hrs → 5 min |
| `/ppc-brief` | Full Amazon PPC campaign structure + bid math | 4 hrs → 5 min |
| `/launch-sequence` | 5-email product launch campaign | 6 hrs → 10 min |
| `/supplier-outreach` | Factory/Alibaba outreach that gets responses | 45 min → 2 min |
| `/product-launch-ad` | Meta, TikTok + Google ad copy variants | 2 hrs → 5 min |
| `listing-optimizer` agent | Full listing audit + rewrite with diff summary | 3 hrs → 10 min |

## Installation

### Option 1: Via Claude Code Marketplace
```bash
/plugin marketplace add [your-github-username]/ecommerce-seller-plugin
/plugin install ecommerce-seller-toolkit
```

### Option 2: Manual Install
```bash
# Clone to your Claude skills directory
git clone https://github.com/[your-username]/ecommerce-seller-plugin ~/.claude/plugins/ecommerce-seller-toolkit

# Verify install
/seller-help
```

## Quick Start Examples

### Write a product listing in 60 seconds
```
/product-listing "Bamboo Cutting Board" | "juice groove, non-slip feet, eco-certified, 18x12 inch" | "health-conscious home cooks" | "bamboo cutting board, wood cutting board, large cutting board kitchen" | amazon
```

### Respond to a 1-star review
```
/review-response 1 | "Broke after 2 uses. Total waste of money." | kitchen gadget | EcoKitchen
```

### Build a full PPC launch campaign
```
/ppc-brief "Bamboo Cutting Board" | $600 | 28% | launch | "bamboo cutting board, kitchen cutting board, large wood cutting board, eco cutting board, professional cutting board"
```

### Create a 5-email launch sequence
```
/launch-sequence "Bamboo Cutting Board" | April 15 | "eco-conscious home cooks, 30–55" | 25% | EcoKitchen
```
